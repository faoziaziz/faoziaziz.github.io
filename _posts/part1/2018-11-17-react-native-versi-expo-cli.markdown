Pertama yang harus diinstall adalah expo-cli

```bash
sudo npm install -g expo-cli
```

lalu buat project folder dengan nama natKu

```bash
expo init natKu
```
masuk ke direktory natKu, dan jalankan programmnya

```bash
cd natKu
npm start
```
